
# ¿Cuándo y por qué usar GIT y GitHub?

Es importante usar GIT en todo proyecto porque me permite tener historial de los cambios, interactuar con otros en la construccion del mismo proyecto y crear versiones segun la actividad que se esta realizando con el uso de las ramas o la recuperacion de informacion cuando se generan errores

Github o cualquier otro repostirio de codigo (Gitlab, Bitbucket, etc) es importante su uso para guardar la informacion del repositorio remotamente y tener la posibilidad de accesarlo entre varios miembros del equipo, y crear procesos de automatizacion para su despliegue entre otros
